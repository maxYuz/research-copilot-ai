---

## 研究背景

**产品**：某在线学习平台
**研究目的**：了解用户学习体验优化方向
**研究方法**：深度访谈（5人）+ 问卷调查（50份）

---

## 定性发现（访谈）

### 共性问题

1. **时间碎片化问题**
   - 3/5用户表示"没有完整的学习时间"
   - 用户典型场景：通勤、午休、哄娃入睡后
   - 用户原话："每次想学习的时候不是娃醒了就是该睡了"

2. **课程选择困难**
   - 4/5用户表示"不知道该选什么课"
   - 用户原话："课程太多了，根本不知道怎么选，挑着挑着就不想学了"
   - 用户期望：有学习路径推荐或能力测评

3. **缺乏学习动力**
   - 5/5用户表示"一个人学习太孤单"
   - 用户原话："有同事一起学可能更有动力，但现在就是自己学，很枯燥"
   - 用户期望：社群功能或学习伙伴

4. **学习效果不明确**
   - 4/5用户表示"学完不知道记住了多少"
   - 用户原话："看视频的时候觉得懂了，但第二天就忘了"
   - 用户期望：有测验或实践作业来检验学习效果

### 个性化需求

- 新手用户：需要入门引导和基础课程
- 进阶用户：需要针对性的技能提升
- 转行用户：需要完整的职业路径规划

## 定量发现（问卷）

### 使用行为

| 使用频率 | 占比 |
|----------|------|
| 每天 | 15% |
| 每周几次 | 35% |
| 每月几次 | 30% |
| 偶尔 | 20% |

### 核心痛点

| 问题 | 选择比例 |
|------|----------|
| 没有时间学习 | 72% |
| 不知道学什么 | 58% |
| 学习效果不明显 | 54% |
| 缺乏动力 | 48% |
| 课程价格太贵 | 35% |

### 用户满意度

- 平均满意度：**3.1分**（满分5分）
- NPS：**-8分**（37%贬损者，29%被动者，34%推荐者）

### 愿意付费的功能

| 功能 | 占比 |
|------|------|
| 真人导师指导 | 68% |
| 实战项目练习 | 62% |
| 社群学习 | 45% |
| 证书认证 | 40% |
| 都不愿意 | 5% |
